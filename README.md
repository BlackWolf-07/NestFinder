# NestFinder 🏠

**NestFinder** is a modern, AI-powered real estate discovery platform designed to help users find their perfect homes with ease and trust. It combines a premium user interface with advanced features like AI property assistance, verified listings, and immersive neighborhood insights.

## ✨ Core Features

-   **AI Property Discovery:** Personalized property recommendations and an intelligent chatbot for natural language searching.
-   **Premium UI/UX:** A modern, responsive design built with **Framer Motion** for smooth animations and transitions.
-   **Trust & Verification:** Verified listing badges and an admin approval workflow to ensure quality and prevent fraud.
-   **Community Engagement:** Detailed property reviews, ratings, and a robust reporting system.
-   **Neighborhood Insights:** Key highlights about nearby schools, hospitals, transport, and markets.
-   **Smart Management:** Comprehensive dashboards for buyers, owners, and admins.
-   **Immersive Features:** Dark/Light mode support, interactive maps, and automated PDF brochure generation.

## 🛠️ Tech Stack

-   **Frontend:** React 19, Tailwind CSS 4, Framer Motion, Lucide Icons, Zustand, React Router 7.
-   **Backend:** Node.js, Express, MySQL (with `mysql2` and migrations).
-   **AI Integration:** Google Gemini Pro API.
-   **Payments:** Stripe Integration.
-   **Maps:** Leaflet & OpenStreetMap.
-   **Notifications:** Sonner.
